# yuki

一个根据目录内文件生成 README.md 的小工具

[![作者](https:\//img.shields.io\/badge/%E4%BD%9C%E8%80%85-KyonHuang-7AD6FD.svg)](http:\//kyonhuang.top)

## 正在开发中

#### 需求

找到文件夹下同类型文件，生成 README.md 的目录文件，带有链接。

## 使用方法

```bash
npm install -g yuki
```
